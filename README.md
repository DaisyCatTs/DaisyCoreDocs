# Daisy Docs

Documentation site for the Daisy ecosystem.

## Stack

- Astro
- Starlight
- MDX
- Cloudflare Pages

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run build
npx wrangler pages deploy dist
```
