# DaisyCore

DaisyCore is a modern Paper platform built for Kotlin-first server development. It aims to give one coherent dependency and one coherent way of building the systems you use repeatedly in real servers.

## What DaisyCore includes

- Commands built around clean Kotlin DSLs
- Menus and views with safe runtime behavior
- Scoreboards and tablists designed for high-frequency updates
- Shared text, placeholder, item, and runtime layers
- One platform bootstrap for projects that want the full stack
