<StepCard step={1} title="Use DaisySeries when config values need normalization" product="series">
  <p>Config-heavy Paper values should not be normalized in runtime files with plugin-local helpers.</p>
</StepCard>

<StepCard step={2} title="Parse values before they reach runtime UI code" product="series">
```kotlin
val icon = DaisyMaterials.parse(config.icon)
val clickSound = DaisySounds.parse(config.sound)
val enchantment = DaisyEnchantments.parse(config.enchantment)
val effect = DaisyPotions.parse(config.effect)
```
</StepCard>

<StepCard step={3} title="Let DaisyCore own commands, menus, sidebars, and tablists" product="core" dependency="DaisyCore">
```kotlin
player.openMenu("Profile", rows = 3) {
    background(Material.GRAY_STAINED_GLASS_PANE) {
        name(" ")
    }
}
```
</StepCard>
