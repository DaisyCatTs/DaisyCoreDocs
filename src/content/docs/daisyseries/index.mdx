# DaisySeries

<div class="home-hero">
  <span class="home-kicker">The Lightweight Utility Toolkit</span>
  <h2>Modern Paper-first enum parsing without dragging in a full platform.</h2>
  <p>DaisySeries gives you clean config-safe parsing, canonical lowercase keys, friendly display names, and curated aliases for the values plugins constantly touch.</p>

```kotlin
dependencies {
    implementation("cat.daisy:DaisySeries:0.1.0-SNAPSHOT")
}
```

<div class="value-strip">
  <div class="value-pill"><strong>Starter pack</strong><span>Materials, sounds, and item flags ship first because they remove the most repeated plugin boilerplate immediately.</span></div>
  <div class="value-pill"><strong>Config-safe</strong><span>Use lowercase keys for storage, display names for UI, and curated aliases for practical plugin input.</span></div>
  <div class="value-pill"><strong>Works alone or with DaisyCore</strong><span>Use it as a standalone utility library or pair it with DaisyCore when your plugin also needs runtime systems.</span></div>

## What DaisySeries includes

<div class="home-grid">
  <div class="home-card">
    <h3>Materials</h3>
    <p>Parse config-friendly material names, get canonical keys, and render clean display names for UI.</p>
  </div>
  <div class="home-card">
    <h3>Sounds</h3>
    <p>Resolve modern Paper sound names from spaced, kebab-case, lowercase, or canonical inputs.</p>
  </div>
  <div class="home-card">
    <h3>Item Flags</h3>
    <p>Parse single flags or config lists and keep canonical output stable for serialization.</p>
  </div>
</div>
