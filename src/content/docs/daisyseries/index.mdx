## What DaisySeries handles

<div class="feature-matrix">
  <div class="feature-card"><strong>Materials</strong><p>Config-friendly parsing for item and block selections with stable serialization keys.</p></div>
  <div class="feature-card"><strong>Sounds</strong><p>Canonical sound values and aliases for config-driven audio cues.</p></div>
  <div class="feature-card"><strong>Enchantments and potions</strong><p>Registry-sensitive values with one consistent parser surface.</p></div>
  <div class="feature-card"><strong>Dictionary lookups</strong><p>Static committed dictionaries so values stay searchable and linkable in docs.</p></div>
</div>
