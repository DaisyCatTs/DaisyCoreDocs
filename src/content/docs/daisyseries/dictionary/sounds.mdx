# Sounds Dictionary

Use these values for feedback sounds, config-driven audio cues, and lightweight sound selection.

## Canonical key rules

- Canonical keys are lowercase underscore names like `entity_player_levelup`.
- Curated aliases are accepted on input only.
- Namespaced inputs are normalized when the DaisySeries sound parser accepts them.

## Quick config example

```yaml
feedback_sound: ambient_basalt_deltas_additions
```

## Quick usage example

```kotlin
val value = DaisySounds.parse(config.feedback_sound)
val key = DaisySounds.key(value)
val label = DaisySounds.displayName(value)

val typed = required("feedback_sound", soundCodec())
```
