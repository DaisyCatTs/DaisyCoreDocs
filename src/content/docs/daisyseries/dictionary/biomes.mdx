# Biomes Dictionary

Use these values for biome-aware config, region defaults, and gameplay systems that need stable biome selection.

## Canonical key rules

- Canonical keys are lowercase underscore names like `cherry_grove`.
- Curated aliases are accepted on input only.
- Namespaced inputs like `minecraft:cherry_grove` are normalized on parse.

## Quick config example

```yaml
biome: badlands
```

## Quick usage example

```kotlin
val value = DaisyBiomes.parse(config.biome)
val key = DaisyBiomes.key(value)
val label = DaisyBiomes.displayName(value)

val typed = required("biome", biomeCodec())
```
