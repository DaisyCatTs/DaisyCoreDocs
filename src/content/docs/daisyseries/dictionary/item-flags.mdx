# Item Flags Dictionary

Use these values for config-driven item metadata hiding and item builder settings.

## Canonical key rules

- Canonical keys are lowercase underscore names like `hide_enchants`.
- Item flags currently do not ship curated aliases.
- Use `parseMany(...)` or `itemFlagsCodec()` when reading a set of flags.

## Quick config example

```yaml
flags: hide_additional_tooltip
```

## Quick usage example

```kotlin
val value = DaisyItemFlags.parse(config.flags)
val key = DaisyItemFlags.key(value)
val label = DaisyItemFlags.displayName(value)

val typed = required("flags", itemFlagsCodec())
```
