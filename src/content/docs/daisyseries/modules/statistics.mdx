# Statistics

`DaisyStatistics` is the statistic entrypoint.

## What it does

- parses normalized statistic input
- exposes canonical lowercase keys
- exposes friendly display names

```kotlin
val statistic = DaisyStatistics.parse("player kills")
val key = DaisyStatistics.key(statistic)
val label = DaisyStatistics.displayName(statistic)
```
