# YAML Loading

Main APIs:

- `DaisyYaml.load(file, codec)`
- `DaisyYaml.handle(file, codec)`
- `plugin.loadYamlConfig(name, codec)`
- `plugin.yamlConfigHandle(name, codec)`
- `plugin.ensureDefaultConfigResource(...)`
