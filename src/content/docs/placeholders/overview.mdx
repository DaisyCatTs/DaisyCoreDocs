# Placeholders Overview

The placeholder layer defines how viewer-aware values are resolved and reused across scoreboards, tablists, menus, and command responses.

## Why this layer exists

Without a shared placeholder system, every feature ends up with its own ad-hoc value substitution. DaisyCore keeps that logic in one place so renderers can stay simple.

## Typical uses

- player-aware sidebar lines
- tablist footer values
- menu item text
- command reply formatting
