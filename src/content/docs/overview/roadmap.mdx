# Roadmap

## Current products

- **DaisyCore**: the Paper platform product
- **DaisySeries**: the lightweight parsing and enum utility product

## DaisyCore

DaisyCore is focused on:

- platform bootstrap
- commands
- menus
- shared text and placeholders
- items
- sidebars
- tablists

See the platform docs: [DaisyCore Overview](/daisycore/)

## DaisySeries

The first DaisySeries starter pack ships:

- materials
- sounds
- item flags

Planned later:

- enchantments
- potions

See the utility docs: [DaisySeries Overview](/daisyseries/)
