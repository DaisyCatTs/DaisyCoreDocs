# Text Overview

The text layer is Adventure-first. It provides the standard message and rendering model used by commands, menus, scoreboards, and tablists.

<div class="reference-grid">
  <div class="reference-panel">
    <h3>Why it exists</h3>
    <p>DaisyCore keeps one canonical text model so its systems do not drift into separate string-only APIs.</p>
  </div>
  <div class="reference-panel">
    <h3>What to expect</h3>
    <p>Adventure components stay central, while DaisyCore renderers keep feature-specific contexts clean.</p>
  </div>
</div>
