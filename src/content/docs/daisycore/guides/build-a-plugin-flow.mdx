<StepCard step={1} title="Bootstrap DaisyCore once" product="core">
  <p>You can build this entire flow with direct MiniMessage strings. Shared config-backed text is the scaling path, not the requirement.</p>

```kotlin
class ProfilePlugin : JavaPlugin() {
    lateinit var daisy: DaisyPlatform
        private set

    override fun onEnable() {
        daisy =
            DaisyPlatform.create(this) {
                commands()
                menus()
                scoreboards()
                tablists()
            }
    }

    override fun onDisable() {
        daisy.close()
    }
}
```
</StepCard>

<StepCard step={2} title="Build sidebar and tablist with direct MiniMessage strings" product="core">
```kotlin
fun profileSidebar(player: Player) =
    sidebar {
        title("<gradient:#7dd3fc:#c4b5fd>Profile</gradient>")
        line("coins") {
            text("<gray>Coins: <white>1,250</white>")
        }
    }

fun profileTablist(player: Player) =
    tablist {
        header("<gradient:#7dd3fc:#c4b5fd>Welcome back</gradient>\n<white>${player.name}</white>")
        footer("<gray>Coins: <white>1,250</white>")
    }
```
</StepCard>

<StepCard step={3} title="Connect everything inside one command group" product="core">
```kotlin
@DaisyCommandSet
object ProfileCommands : DaisyCommandGroup({
    command("profile") {
        description("Open your profile tools")

        player {
            reply("<gradient:#7dd3fc:#c4b5fd>Opening your profile.</gradient>")

            player.openMenu("Profile", rows = 3) {
                background(Material.GRAY_STAINED_GLASS_PANE) {
                    name(" ")
                }

                slot(13) {
                    item(Material.PLAYER_HEAD) {
                        name("<gradient:#7dd3fc:#c4b5fd>${player.name}</gradient>")
                        lore(
                            "<gray>Inline Daisy menu flow</gray>",
                            "<white>Built for ${player.name}</white>",
                        )
                    }

                    message("<gray>Clicked the profile card.</gray>")
                    closeOnClick()
                }
            }

            daisy.scoreboards?.show(player, profileSidebar(player))
            daisy.tablists?.show(player, profileTablist(player))
        }
    }
})
```
</StepCard>

<StepCard step={4} title="Move to shared text only when the plugin is config-heavy" product="config" dependency="DaisyConfig">
  <p>If your plugin keeps wording in config or lang files, swap local strings for `replyLang(...)`, `nameLang(...)`, `titleLang(...)`, and related helpers. Those config paths are useful scaling tools, not DaisyCore requirements.</p>
</StepCard>
