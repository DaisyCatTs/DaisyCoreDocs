# Bootstrap API

<div class="doc-lead">
  <span class="eyebrow">Reference</span>
  <h3>DaisyPlatform is the single bootstrap surface for DaisyCore runtime services.</h3>
</div>

```kotlin
val daisy =
    DaisyPlatform.create(plugin) {
        text()
        placeholders()
        items()
        scoreboards()
        tablists()
        commands()
        menus()
    }
```

## Shutdown

```kotlin
override fun onDisable() {
    daisy.close()
}
```
