# Items Overview

The item layer is where shared item builders, item naming, and future template support live.

It exists so menu code does not need to reinvent item creation every time a page renders or refreshes.

## What this layer is for

- creating inventory items cleanly
- applying shared naming and lore styles
- keeping menu rendering readable
